elementsJS.io

This is a very extensive DOM Manipulation/Utility Function JavaScript Library. It features chainable methods and a new syntax that is usable with a gulp or grunt preprocessor plugin. It's aim is to make coding more efficient and fun and code more readable. The project site itself, I worked very hard on, and I hope it shows. It is a single-page app written without a Framework, just HTML5, CSS3 and good ol' JavaScript! It is hosted on GitHub.
