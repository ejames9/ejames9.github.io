elementsJS.io

This is a very extensive DOM Manipulation/Utility Function JavaScript Library. It features chainable methods and a new syntax that is usable with a gulp or grunt preprocessor plugin. It's aim is to make coding more efficient and fun and code more readable. The project site itself, I worked very hard on, and I hope it shows. It is a single-page app written without a Framework, just HTML5, CSS3 and good ol' JavaScript! It is hosted on GitHub.

elementsJS.io Nav-Menu

This is the Side-Navigation Menu for the elementsjs.io project site. I am featuring this as a separate project, because it took a lot of work to create, and I think the result is very intuitive yet sophisticated. Please visit the site and take it for a spin!

gulp-elementsJS-Interpreter

This is just a screenshot of the npm page, where the elementsJS library interpreter gulp-plugin is published. npm doesn't allow it's pages to load in an iframe, so visit the site if you'd like a closer look. The interpreter is some of the more complex code I've written. Of course, by now I'd have written it differently, but, what are ya gonna do?
